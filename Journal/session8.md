#BOM + JLCPCB order

fabrication time for the board. exported all the outputs from kicad, grabbed the gerbers, bom, cpl, the whole thing. opened the cpl csv in sheets. ipped everything into a “production” folder and tossed it into jlcpcb’s quote page, turned on pcba so they actually assemble it. uploaded the bom + cpl, fixed the missing / wrong parts, especially the 27 ohm resistors and flash chip part number, and made sure headers are not selected for assembly. tweaked parts to use as many basic ones as possible so the price doesn’t blow up. then checked the 3d view / placement, nudged usb‑c, ldo, and flash so they line up right and match the markings, and saved to cart
